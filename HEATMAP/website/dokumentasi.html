Developer API Documentation

Integrasi pembayaran QRIS Dinamis (DANA) otomatis.

1. Kredensial API

Setiap request ke API wajib menyertakan Header berikut:

x-api-key
id_merchant
Base URL https://klikqris.com/api
POST Create Transaction

Membuat tagihan QRIS baru.

Endpoint: /qris/create

Penting: Kode unik mungkin ditambahkan otomatis. Gunakan total_amount dari response JSON untuk ditampilkan ke user.
Body Parameters (JSON)
order_id * String. ID unik transaksi Anda (misal: INV-123).
amount * Integer. Nominal bayar.
id_merchant * String. ID Merchant Anda.
keterangan String. Catatan transaksi (Opsional).
PHP cURL REQUEST 
$curl = curl_init();

curl_setopt_array($curl, array(
  CURLOPT_URL => 'https://klikqris.com/api/qris/create',
  CURLOPT_RETURNTRANSFER => true,
  CURLOPT_CUSTOMREQUEST => 'POST',
  CURLOPT_POSTFIELDS =>'{
    "order_id": "INV-123",
    "id_merchant":176933217914,
    "amount": 1600,
    "keterangan": "Pembayaran Invoice #123"
}',
  CURLOPT_HTTPHEADER => array(
    'Content-Type: application/json',
    'x-api-key: OHJP3qz0RthcVAQ0XhrVbdVcoDggmpBQGqKnvMUX',
    'id_merchant: 176933217914'
  ),
));

$response = curl_exec($curl);

    }
}
POST Webhook / Callback URL

Sistem kami akan mengirimkan data JSON secara otomatis ke server Anda ketika status pembayaran berubah menjadi SUCCESS (PAID).

Setup: Masuk ke menu Settings di dashboard untuk memasukkan URL Callback Anda.
Payload yang dikirim ke Server Anda:
JSON PAYLOAD 
{
    "order_id": "DIRECT-176835469862-8460-202601252147",
    "status": "PAID",
    "amount": 1000,
    "total_amount": 1215,
    "payment_date": "2026-01-25 21:48:01",
    "created_at": "2026-01-25 21:47:42",
    "updated_at": "2026-01-25 21:48:01",
    "keterangan": "Pembayaran Paket A",
    "direct_url": "https://klikqris.com/payqris/176835469862/INV-123456"
}

*Server Anda harus merespon dengan HTTP Code 200 OK agar sistem tidak mencoba mengirim ulang.
